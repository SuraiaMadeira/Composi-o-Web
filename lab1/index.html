Dicas HTML

  • A imagem do header foi criada em https://www.canva.com/
  • Podem obter uma palete de cores através de aplicações online, como por exemplo hp://www.colourco.de/
  • O Menu deverá ser feito através de uma lista não ordenada. Para ter o aspeto pretendido devem atribuir os seguintes eslos:
    • display: inline
    • list-style-type: none
    • text-decoraon:none
  • Aprender HTML / Dicas e os Tutoriais deverão estar inseridos em uma tabela 2x2. A coluna Dicas deverá ocupar 70% da largura da página. A coluna Tutoriais deve ocupar 2 linhas
  • Os Tutoriais são iframes com vídeos do youtube
  • O formulário de contacto está inserido num div. Deverá atribuir ao div os seguintes eslos:
    • background-color: whitesmoke
    • padding-le:100
    • padding-top:30
  • Os inputs do formulário deverão ter uma class associada que tenha a seguinte configuração:
    • width: 40%
    • line-height: 30px
